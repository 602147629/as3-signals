Classpublic class Signal
ImplementsISignal
SubclassesNativeRelaySignal

Signal dispatches events to multiple listeners. It is inspired by C# events and delegates, and by signals and slots in Qt. A Signal adds event dispatching functionality through composition and interfaces, rather than inheriting from a dispatcher.

Project home: http://github.com/robertpenner/as3-signals/
